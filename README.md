# Manus CRM (Starter)

This is the starter code for Manus Voice Pro CRM.

## Quick local start

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
node server.js

## Deploy
- Create Supabase project and run SQL schema (see /database/schema.sql)
- Create Vercel project and link this repo
- Add environment variables from .env.template
